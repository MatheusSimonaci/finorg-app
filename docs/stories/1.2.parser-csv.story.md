## User Story

**Como** usuário,
**Quero** importar meu extrato CSV do Nubank, XP ou Bitybank,
**Para que** o sistema entenda o formato de cada banco automaticamente sem configuração manual.

## Dev Notes

### Formato CSV do Nubank (conta corrente)

```csv
Data,Descrição,Valor
2026-03-01,IFOOD,45.90
2026-03-01,SALARIO EMPRESA,5000.00
```
- Valores negativos = saída de dinheiro (gasto)
- Valores positivos = entrada (receita)

### Formato CSV do Nubank (cartão de crédito)

```csv
date,title,amount
2026-03-01,iFood*Restaurante,-45.90
2026-03-01,Pagamento recebido,500.00
```

### Formato XP (extrato de custódia)

```csv
Data,Histórico,Crédito,Débito,Saldo
01/03/2026,Compra BOVA11,,152.00,9848.00
01/03/2026,Dividendos MXRF11,12.50,,9860.50
```

### Função de hash para deduplicação

```typescript
import { createHash } from 'crypto'

function generateTransactionHash(date: string, description: string, amount: number): string {
  return createHash('sha256')
    .update(`${date}::${description}::${amount}`)
    .digest('hex')
    .slice(0, 16)
}
```
