## User Story

**Como** usuário,
**Quero** que as transações importadas sejam classificadas automaticamente pela IA,
**Para que** eu não precise categorizar cada gasto manualmente toda vez que importar um extrato.

- [ ] AC10: Transações de contas XP e Bitybank classificadas automaticamente como `nature=pessoal, category=investimento` sem chamar a API

## Dev Notes

### Prompt do Sistema (contexto do usuário)

```typescript
const SYSTEM_PROMPT = `
Você é um classificador financeiro pessoal especializado.

CONTEXTO DO USUÁRIO:
- O usuário é pessoa física que usa a conta Nubank para gastos pessoais E empresariais misturados
- Contas XP e Bitybank são EXCLUSIVAMENTE investimentos pessoais
- O usuário tem uma empresa e usa ferramentas profissionais pagas

REGRAS DE CLASSIFICAÇÃO (aplicar antes da sua análise):
- CapCut, Adobe, Figma, Canva, Notion, Slack, Linear → nature: work_tool, category: assinatura
- Qualquer transferência recebida com "SALARIO" ou "FOLHA" → nature: pessoal, category: receita, type: receita
- Transferências enviadas para "MEI", "CNPJ", nome da empresa → nature: empresa

DIMENSÕES DE CLASSIFICAÇÃO:
- nature: pessoal | empresa | work_tool | misto
- category: saude | educacao | lazer | alimentacao | moradia | assinatura | investimento | transporte | receita | pet | servicos | outros
- type: gasto | investimento | reserva | receita | transferencia

Responda SEMPRE em JSON válido. Confidence de 0.0 a 1.0.
Se confidence < 0.7, explique o motivo da incerteza em "reasoning".
`
```

### Schema de resposta da IA

```typescript
type ClassificationResult = {
  nature: 'pessoal' | 'empresa' | 'work_tool' | 'misto'
  category: string
  subcategory?: string
  type: 'gasto' | 'investimento' | 'reserva' | 'receita' | 'transferencia'
  confidence: number
  reasoning?: string
}

// Payload enviado por transação
type TransactionPayload = {
  id: string
  date: string
  description: string
  amount: number
  account: string  // nubank | xp | bitybank
}
```

### Estratégia de batching

```typescript
async function classifyBatch(transactions: TransactionPayload[]): Promise<ClassificationResult[]> {
  // Dividir em grupos de 20
  const batches = chunk(transactions, 20)
  const results = []
  
  for (const batch of batches) {
    const response = await openai.chat.completions.create({
      model: 'gpt-4o-mini',
      response_format: { type: 'json_object' },
      messages: [
        { role: 'system', content: SYSTEM_PROMPT },
        { role: 'user', content: JSON.stringify(batch) }
      ]
    })
    results.push(...parseResponse(response))
    // Delay para evitar rate limit
    await sleep(500)
  }
  
  return results
}
```

### Custo estimado (GPT-4o-mini)

- Input: ~$0.15/1M tokens
- Output: ~$0.60/1M tokens
- Estimativa: 50 transações ≈ $0.01 ≈ R$0.06
