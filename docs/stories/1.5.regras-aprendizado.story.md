## User Story

**Como** usuário,
**Quero** que minhas correções de classificação gerem regras permanentes,
**Para que** importações futuras sejam classificadas corretamente sem revisão manual repetitiva.

---

## Acceptance Criteria

- [x] AC1: Ao corrigir uma classificação, o sistema apresenta modal para criar regra permanente
- [x] AC2: Regras são salvas com: `pattern`, `nature`, `category`, `type`, `subcategory`
- [x] AC3: Regras têm prioridade total sobre a classificação IA
- [x] AC4: Tela de configurações exibe lista de todas as regras com opção de editar e deletar
- [x] AC5: Regras pré-configuradas do sistema carregadas no seed inicial
- [x] AC6: Sistema sugere `pattern` baseado na descrição da transação
- [x] AC7: Regras do tipo `regex` são suportadas
- [x] AC8: Indicador na tela de revisão mostrando se a classificação veio de uma regra ou da IA

### Aplicação de regras

```typescript
function applyRules(tx: NormalizedTransaction, rules: ClassificationRule[]): ClassificationResult | null {
  for (const rule of rules) {
    const regex = new RegExp(rule.pattern, 'i')
    if (regex.test(tx.description)) {
      return {
        nature: rule.nature,
        category: rule.category,
        type: rule.type,
        confidence: 1.0,
        reasoning: `Regra: "${rule.pattern}"`
      }
    }
  }
  return null
}
```
