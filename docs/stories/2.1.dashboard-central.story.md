## Dev Notes

### Cálculo de renda líquida pessoal

```typescript
async function getPersonalNetIncome(month: string): Promise<number> {
  const transactions = await db.transaction.findMany({
    where: {
      date: { gte: startOfMonth(month), lte: endOfMonth(month) },
      type: 'receita',
      nature: 'pessoal',
      isReimbursable: false
    }
  })
  return transactions.reduce((sum, tx) => sum + Number(tx.amount), 0)
}
```

### API Route do dashboard

`GET /api/dashboard?month=2026-03` → retorna todos os KPIs em uma única chamada.
