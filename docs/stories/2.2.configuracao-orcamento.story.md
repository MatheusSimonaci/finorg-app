## User Story

**Como** usuário,
**Quero** definir qual percentual da minha renda líquida pessoal deve ir para cada categoria de gasto,
**Para que** eu tenha um plano financeiro personalizado como base de comparação.

---

## Acceptance Criteria

- [ ] AC1: Tela de configuração lista todas as categorias com campo numérico de % editável
- [ ] AC2: Soma dos percentuais exibida em tempo real com indicador verde (≤ 100%) ou vermelho (> 100%)
- [ ] AC3: Categorias: moradia, alimentação, transporte, saúde, educação, lazer, assinaturas, pet, serviços, outros
- [ ] AC4: Categoria "investimentos" aparece mas é marcada como "gerenciada no módulo de investimentos"
- [ ] AC5: Template inicial sugerido baseado na regra 50/30/20 com botão "Aplicar template"
- [ ] AC6: Configuração persistida no banco (tabela `Budget`) e aplicada a todos os meses futuros por padrão
- [ ] AC7: Campo "alerta em X% do limite" por categoria (padrão: 80%)
- [ ] AC8: Botão "Salvar" com feedback de sucesso
