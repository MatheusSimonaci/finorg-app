## User Story

## Dev Notes

### Cálculo do limite por categoria

```typescript
function calculateLimit(category: string, personalIncome: number, rules: Budget[]): number {
  const rule = rules.find(r => r.category === category)
  if (!rule) return 0
  return (personalIncome * rule.targetPct) / 100
}
```
