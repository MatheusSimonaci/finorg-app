## User Story

**Como** usuário,
**Quero** configurar qual percentual de cada tipo de ativo devo ter no portfólio por modo,
**Para que** o sistema saiba quando estou dentro ou fora do planejado em cada fase financeira.

---

## Acceptance Criteria

- [ ] AC1: Tela com sliders/inputs por tipo de ativo para o modo "normal"
- [ ] AC2: Soma total dos targets validada em tempo real — deve totalizar 100%
- [ ] AC3: Suporte a targets diferentes por modo: normal | sonho_ativo | rebalanceando
- [ ] AC4: Template sugerido (perfil moderado) com distribuição padrão, aplicável com um clique
- [ ] AC5: Histórico de mudanças de target salvo com data (auditoria)
- [ ] AC6: Modo "sonho_ativo" sugere automaticamente mais % em conta_remunerada e tesouro_selic
