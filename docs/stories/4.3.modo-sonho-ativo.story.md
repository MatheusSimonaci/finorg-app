## User Story

**Como** usuário,
**Quero** que quando um sonho estiver ativo, o sistema ajuste a alocação-alvo recomendada da carteira,
**Para que** eu naturalmente concentre mais liquidez nos ativos certos para realizar o sonho.

## Dev Notes

### Lógica de alocação modo sonho

Quando um sonho está ativo, o `AllocationTarget` para o modo `sonho_ativo` é sugerido com base no `targetAmount` do sonho vs. o portfólio total:

```typescript
function suggestDreamAllocation(dreamAmount: number, portfolioTotal: number, normalTargets: AllocationTarget[]): AllocationTarget[] {
  const dreamPct = Math.min((dreamAmount / portfolioTotal) * 100, 40) // máx 40% em liquidez
  
  return normalTargets.map(t => {
    if (['conta_remunerada', 'tesouro'].includes(t.assetType)) {
      return { ...t, targetPct: t.targetPct + dreamPct / 2 }
    }
    // Reduz proporcionalmente os demais
    return { ...t, targetPct: t.targetPct * (1 - dreamPct / 200) }
  })
}
```
