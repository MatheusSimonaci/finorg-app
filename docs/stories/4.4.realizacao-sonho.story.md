## User Story

**Como** usuário,
**Quero** registrar quando realizei um sonho,
**Para que** o sistema atualize meu portfólio, registre a conquista e inicie automaticamente o próximo ciclo.

---

## Acceptance Criteria

- [ ] AC1: Botão "Realizar sonho 🏆" disponível quando progresso ≥ 90% do valor-alvo
- [ ] AC2: Modal de confirmação: valor sacado (R$), data, de quais ativos sairá o dinheiro, observação opcional
- [ ] AC3: Ao confirmar: valores dos ativos afetados são atualizados (reduzidos pelo valor sacado)
- [ ] AC4: Status do sonho muda para REALIZADO com `achievedAt = hoje`
- [ ] AC5: Tela de celebração: "🎉 Parabéns! Você realizou [nome do sonho]!" com histórico de quando foi criado vs. realizado
- [ ] AC6: Próximo sonho na fila (maior prioridade não arquivado) promovido automaticamente para ACUMULANDO
- [ ] AC7: Estado da carteira muda para REBALANCEANDO (sem alerta de erro — apenas informativo)
- [ ] AC8: Página de histórico de sonhos realizados com linha do tempo

## Dev Notes

### Fluxo de realização

```typescript
async function realizeDream(dreamId: string, withdrawal: { assetId: string, amount: number }[]) {
  await db.$transaction(async (tx) => {
    // 1. Atualizar ativos
    for (const w of withdrawal) {
      await tx.asset.update({ where: { id: w.assetId }, data: { currentValue: { decrement: w.amount } } })
      await tx.assetSnapshot.create({ data: { assetId: w.assetId, month: currentMonth(), value: newValue } })
    }
    
    // 2. Marcar sonho como realizado
    await tx.dream.update({ where: { id: dreamId }, data: { status: 'realizado', achievedAt: new Date() } })
    
    // 3. Promover próximo sonho
    const next = await tx.dream.findFirst({ where: { status: 'planejando' }, orderBy: { priorityOrder: 'asc' } })
    if (next) await tx.dream.update({ where: { id: next.id }, data: { status: 'acumulando' } })
  })
}
```
