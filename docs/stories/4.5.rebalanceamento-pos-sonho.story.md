## User Story

**Como** usuário,
**Quero** ver como minha carteira está progredindo no rebalanceamento após realizar um sonho,
**Para que** eu saiba quando voltei ao equilíbrio ideal para focar no próximo objetivo.

## Dev Notes

### Cálculo de progresso de rebalanceamento

```typescript
function computeRebalancingProgress(currentAllocation: number[], targetAllocation: number[]): number {
  const totalDeviation = currentAllocation.reduce((sum, curr, i) => {
    return sum + Math.abs(curr - targetAllocation[i])
  }, 0)
  
  // totalDeviation = 0 → rebalanced (100%)
  // Assume initial deviation was captured at dream realization
  const initialDeviation = getInitialDeviation() // salvo no momento do saque
  if (initialDeviation === 0) return 100
  
  return Math.max(0, Math.round((1 - totalDeviation / initialDeviation) * 100))
}
```
