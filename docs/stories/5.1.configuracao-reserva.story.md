## User Story

**Como** usuário,
**Quero** configurar quantos meses de gastos minha reserva deve cobrir e como o valor é calculado,
**Para que** eu tenha uma meta clara e personalizada para a minha realidade.

---

## Acceptance Criteria

- [x] AC1: Campo configurável: número de meses alvo (padrão: 4, range: 1–24)
- [x] AC2: Janela de cálculo: usar média dos últimos 1, 3 ou 6 meses de gastos pessoais — selecionável
- [x] AC3: Toggle: excluir meses com gastos atípicos (outliers — acima de 2x a média) do cálculo
- [x] AC4: Valor-alvo calculado automaticamente e exibido: "Meta: R$X (4 × R$Y/mês de média)"
- [x] AC5: Valor-alvo atualizado automaticamente quando novos dados são importados
