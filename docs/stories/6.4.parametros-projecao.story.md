## Dev Notes

### Schema Prisma
```prisma
model ProjectionParams {
  id        String   @id @default(cuid())
  assetCategory String  // "tesouro_selic" | "cdb" | "fii" | "acoes" | "cripto" | "outro"
  annualRate    Float    // taxa a.a. decimal (ex: 0.105)
  irRate        Float    // alíquota IR (ex: 0.15)
  updatedAt     DateTime @updatedAt
}

model GlobalParams {
  id            String @id @default("singleton")
  inflationRate Float  @default(0.045)
}
```
