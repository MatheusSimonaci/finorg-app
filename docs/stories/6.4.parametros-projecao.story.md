## User Story

**Como** usuário,
**Quero** configurar as taxas de rentabilidade e inflação usadas nas projeções,
**Para que** as estimativas reflitam minha realidade e expectativas de mercado.

---

## Acceptance Criteria

- [ ] AC1: Formulário de taxas por categoria de ativo com valores padrão pré-carregados:
  - Tesouro Selic: 10.5% a.a.
  - CDB (CDI): 10.5% a.a.
  - FII: 9.6% a.a. (0.8%/mês)
  - Ações: 12% a.a.
  - Cripto: 20% a.a.
- [ ] AC2: Campo de inflação anual (padrão: 4.5%)
- [ ] AC3: Configuração de imposto: IR regressivo para RF (22.5% → 15%), 15% para RV
- [ ] AC4: Botão "Restaurar padrões" retorna todos os valores para os defaults
- [ ] AC5: Parâmetros salvos em tabela `projection_params` no SQLite
- [ ] AC6: Alteração nos parâmetros invalida cache de projeções (recalcula na próxima consulta)

## Dev Notes

### Schema Prisma
```prisma
model ProjectionParams {
  id        String   @id @default(cuid())
  assetCategory String  // "tesouro_selic" | "cdb" | "fii" | "acoes" | "cripto" | "outro"
  annualRate    Float    // taxa a.a. decimal (ex: 0.105)
  irRate        Float    // alíquota IR (ex: 0.15)
  updatedAt     DateTime @updatedAt
}

model GlobalParams {
  id            String @id @default("singleton")
  inflationRate Float  @default(0.045)
}
```
