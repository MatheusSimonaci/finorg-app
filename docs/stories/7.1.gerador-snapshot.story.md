## User Story

**Como** usuário,
**Quero** gerar um snapshot estático do meu dashboard financeiro com um clique,
**Para que** eu tenha uma versão offline e segura para consultar no celular sem expor o sistema principal.

## Dev Notes

### Estratégia de geração
```typescript
// lib/snapshot/generator.ts
// 1. Buscar todos os dados necessários via queries Prisma
// 2. Serializar para JSON embarcado no HTML (sem fetch externo)
// 3. Escrever /snapshot/index.html com template auto-contido
// 4. CSS e charts via SVG inline (sem CDN)

async function generateSnapshot(outputDir: string): Promise<void> {
  const data = await collectSnapshotData(); // queries Prisma
  const html = renderSnapshotTemplate(data);
  await fs.writeFile(path.join(outputDir, 'index.html'), html, 'utf-8');
}
```

### API route
```
POST /api/snapshot/generate
Response: { path: "/snapshot/index.html", generatedAt: ISO_DATE }
```
