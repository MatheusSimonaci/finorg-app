## User Story

**Como** usuário,
**Quero** publicar meu snapshot na Vercel com um único clique,
**Para que** eu possa acessá-lo de qualquer dispositivo via URL sem configurar servidores.

---

## Acceptance Criteria

- [ ] AC1: Tela de configuração para inserir Vercel API Token com campo de senha (masked)
- [ ] AC2: Token armazenado criptografado localmente (não em plaintext no SQLite)
- [ ] AC3: Botão "🚀 Deploy para Vercel" no dashboard (visível apenas se token configurado)
- [ ] AC4: Deploy envia conteúdo de `/snapshot/` via Vercel Deploy API
- [ ] AC5: Após deploy bem-sucedido: exibir URL de acesso público, copiável com 1 clique
- [ ] AC6: Feedback de progresso durante upload (estimativa de tempo)
- [ ] AC7: Tratamento de erros: token inválido, falha de rede, quota excedida — mensagens claras
