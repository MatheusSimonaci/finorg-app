## User Story

**Como** usuário,
**Quero** controlar quem pode acessar meu snapshot quando publicado na Vercel,
**Para que** eu possa compartilhar com segurança ou manter privado conforme necessário.

---

## Acceptance Criteria

- [x] AC1: Configuração de visibilidade: **Público** (qualquer pessoa com URL) ou **Protegido por senha**
- [x] AC2: Modo Protegido: ao acessar o snapshot, exibir formulário de senha antes do conteúdo
- [x] AC3: Senha armazenada como hash bcrypt — nunca em plaintext
- [x] AC4: Aviso exibido SEMPRE antes de gerar/fazer deploy: "Este snapshot contém informações financeiras pessoais. Certifique-se de que a configuração de privacidade está correta."
- [x] AC5: Opção de excluir dados sensíveis do snapshot: ocultar valores numéricos, exibir apenas percentuais e tendências
